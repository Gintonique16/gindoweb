🗺️ LỘ TRÌNH MASTER WEB ĐẸP BẰNG SVELTE + SVELTEKIT – DÀNH RIÊNG CHO NGÀI GIN

⛩️ GIAI ĐOẠN 1 – NỀN TẢNG BẤT DIỆT (Tuần 1–2)

🎯 Mục tiêu:

  • Làm chủ HTML, CSS, JS cơ bản
  • Biết làm web tĩnh responsive

📚 Kiến thức:

  • HTML5: semantic tag, form, media
  • CSS3: Flexbox, Grid, animation, media query
  • JS: event, function, object, DOM

🛠 Công cụ:

  • VSCode + extensions (Prettier, Emmet, Tailwind IntelliSense)
  • Git + GitHub + GitHub Pages (deploy)

🧪 Bài tập:

  • CV cá nhân responsive
  • Landing page có hiệu ứng scroll

🌱 GIAI ĐOẠN 2 – NHẬP MÔN SVELTE (Tuần 3–6)

🎯 Mục tiêu:

  • Làm quen component, reactive, binding
  • Áp dụng Tailwind để style đẹp

📚 Học:

  • props, event, {#if}, {#each}
  • $: reactive variable
  • bind: input, component
  • Store: writable, derived, readable

🧰 Thư viện:

  • Tailwind CSS
  • Iconify hoặc Heroicons
  • Svelte Transition

🧪 Bài tập:

  • Todo app có lưu dữ liệu localStorage
  • Web tĩnh 1 trang (portfolio dạng Svelte component)

🏞️ GIAI ĐOẠN 3 – SVELTEKIT (Tuần 7–10)

🎯 Mục tiêu:

  • Xây dựng site SPA + SSR + SEO
  • Kết hợp Markdown, animation, store

📚 Kiến thức:

  • Routing: +page.svelte, +layout.svelte
  • load(), +page.server.js
  • Markdown: mdsvex, blog tĩnh
  • Form action + validation
  • Dark mode, responsive theme

🧰 Tool:

  • ShadCN UI (chuyển sang Svelte headlessUI)
  • @tailwindcss/typography, lucide-svelte
  • Svelte Motion hoặc GSAP

🧪 Dự án:

  • Blog cá nhân: markdown + theme switcher
  • Landing page kể chuyện: scroll + animation

🦾 GIAI ĐOẠN 4 – MASTERPIECE (Tuần 11+)

🎯 Mục tiêu:

  • Làm sản phẩm thực tế và deploy
  • Tối ưu hiệu năng, SEO, UX

📚 Kiến thức mở rộng:

  • PWA: manifest + offline
  • Lighthouse audit, Web Vitals
  • Cloudflare caching, Netlify/Vercel CI/CD
  • Headless CMS: Sanity, Strapi, Contentful

🧪 Dự án:

  • Webshop demo (store + cart + checkout giả lập)
  • Portfolio showcase: animation đẹp + CV + contact form
  • Blog tĩnh SEO mạnh (SSG)

🌟 GỢI Ý THÊM

  • Tài liệu Svelte: https://svelte.dev
  • SvelteKit: https://kit.svelte.dev
  • Tailwind: https://tailwindcss.com/docs
  • Starter repo: tạo base với vite + svelte + tailwind
